## Edit code with a review step

Select some code, then run **LLMStack: Edit selection with AI…**
(`Cmd/Ctrl+Alt+E`) and describe the change.

LLMStack drafts the edit and shows it as a **native diff**. Nothing touches your
file until you click **Apply**. Changed your mind? Run **LLMStack: Revert last AI
edit** to restore it in one step.

This is the diff → approve → undo loop, running entirely on your machine.
